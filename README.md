# 🌍 Climate Trend Analyzer

## 📌 Overview
A data science project that analyzes historical climate data to identify trends, detect anomalies, and forecast future temperature patterns using machine learning.

---

## 🎯 Problem Statement
Climate change is a major global challenge. This project analyzes climate data to uncover meaningful insights and predict future trends.

---

## 🚀 Features
- 📊 Data Cleaning & Preprocessing
- 📈 Temperature Trend Analysis
- 🚨 Anomaly Detection (Extreme Events)
- 🔮 Future Forecasting using ML
- 🌐 Interactive Streamlit Dashboard

---

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn
- Streamlit

---

## 📊 Project Workflow
Data → Cleaning → EDA → Trend Analysis → Anomaly Detection → Forecasting → Visualization

---

## 📁 Folder Structure


- [app/](app/) → Streamlit dashboard  
- [src/](src/) → Core logic  
- [images/](images/) → Screenshots  
- [outputs/](outputs/) → Generated plots
